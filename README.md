# ApexBuild Construction

A fictional premium construction company website created as a Webleee portfolio demo. The site is built with Astro, Tailwind CSS, static output, and Cloudflare Pages compatibility.

This demo does not claim real clients, certifications, awards, partnerships, or reviews.

## Project Status

- GitHub repo: `https://github.com/zach601/webleee-construction-demo`
- Cloudflare live link: `https://webleee-construction-demo.pages.dev`
- Build status: passed
- Pages created: Home, Services, Projects, About, Contact

## Local Setup

```bash
npm install
npm run dev
```

Open the local URL shown by Astro.

## Build

```bash
npm run build
```

The production output is generated in:

```text
dist
```

## Cloudflare Pages Settings

- Project name: `webleee-construction-demo`
- Framework preset: Astro
- Build command: `npm run build`
- Output directory: `dist`
- Node version: use the Cloudflare Pages default supported Node runtime or set a current LTS runtime in Pages settings.

## Deployment

1. Push this repository to GitHub.
2. Create or connect a Cloudflare Pages project named `webleee-construction-demo`.
3. Set the build command to `npm run build`.
4. Set the output directory to `dist`.
5. Deploy the site.
